## ダウンロード

GitHub Releases の最新リリースから、OS に合うインストーラーを選んでください。

- Windows: `.exe` または `.msi`
- macOS: `.dmg`
- Linux: `.deb`

## 初回起動

- Windows で SmartScreen が出た場合は `USER_QUICK_START.md` を確認してください。
- どのファイルを選べばよいか迷う場合も `USER_QUICK_START.md` を見てください。
